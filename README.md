# Encurtador de Links

#### Passo 1 - Instale as dependências:
```
yarn add
```

#### Passo 2:
Altere o nome do arquivo .env.example para .env e preencha com os dados do seu banco.

#### Passo 3 - Incie a aplicação:
```
yarn start
```
